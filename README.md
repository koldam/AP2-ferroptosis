This repository stores R-code utilized in the article entitled "AP-2 Transcription Factors as Regulators of Ferroptosis: A Family-Wide Profiling in Diverse Cancer Contexts" authored by Damian Kołat, Piotr Gromek, Mateusz Kciuk, Lin-Yong Zhao, Żaneta Kałuzińska-Kołat, Renata Kontek, and Elżbieta Płuciennik. This manuscript is currently under revision stage in the International Journal of Molecular Sciences journal by MDPI.
